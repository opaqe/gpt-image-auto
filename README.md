# GPT 이미지 자동 생성기

> ChatGPT에서 카드뉴스 이미지를 자동으로 순차 생성하는 Chrome 확장 프로그램

📖 **자세한 사용 가이드**: [kkongdon.notion.site/gpt-image-auto-](https://kkongdon.notion.site/gpt-image-auto-)

---

## 주요 기능

- **순차 자동 실행** — 프롬프트를 최대 10개 입력하면 ChatGPT에서 이미지를 하나씩 자동 생성
- **전역 참고 이미지** — 배경·양식 이미지를 등록하면 모든 카드에 자동 첨부 (카드뉴스 일관성 유지)
- **카드별 이미지 첨부** — 카드마다 개별 참고 이미지를 따로 첨부 가능
- **MD 파일 임포트** — `.md` 파일에서 프롬프트를 한 번에 불러오기
- **일괄 다운로드** — 전체 또는 이번 실행분 이미지를 한 번에 저장
- **전역 지침** — 모든 프롬프트 앞에 공통 지침을 자동 삽입
- **모드 설정** — Instant / Thinking / Pro 타임아웃 설정

---

## 설치 방법

> Chrome 브라우저가 필요합니다.

**1. 코드 다운로드**

오른쪽 상단 `Code` → `Download ZIP` 클릭 후 압축 해제

또는 [Releases](../../releases) 탭에서 최신 버전 ZIP 다운로드

**2. Chrome 확장 로드**

```
chrome://extensions/
```
위 주소를 Chrome 주소창에 입력 →

① **개발자 모드** 스위치를 ON

② **"압축 해제된 확장 프로그램 로드"** 클릭

③ 압축 해제한 폴더 선택

**3. 확인**

Chrome 우상단 퍼즐 아이콘 🧩 → `GPT 이미지 자동 생성기` 고정

## 사용 방법

**기본 흐름**

```
① ChatGPT(chatgpt.com) 탭 열기
② 확장 아이콘 클릭 → 팝업 열기
③ 카드별 이미지 프롬프트 입력 (최대 10개)
④ "실행" 버튼 클릭
⑤ ChatGPT에서 이미지 자동 생성 시작
⑥ 완료 후 "이번 실행분 다운로드" 클릭
```

**전역 참고 이미지 사용 (카드뉴스 일관성)**

```
팝업 상단 📎 이미지 등록 → 배경/양식 이미지 업로드
→ 이후 모든 카드에 자동 첨부됨
```

**MD 파일로 프롬프트 일괄 입력**

`.md` 파일에 아래 형식으로 작성 후 "MD 가져오기" 버튼으로 임포트:

```markdown
## 1. 표지
[표지 이미지 프롬프트]

## 2. 본문
[본문 이미지 프롬프트]
```
